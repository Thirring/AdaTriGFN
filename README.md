## Overview

This repository contains the official implementation code for the paper: **"Learning to Defer: Adaptive Multi-View Graph Fusion via Differentiable Three-Way Decision for Span-level Target-Aware Toxicity Extraction"**. 

## Dataset

The dataset used in this research is publicly available. You can access and download it from the following repository:

* **STATEToxiCN Dataset:** [https://github.com/shenmeyemeifashengguo/STATEToxiCN](https://github.com/shenmeyemeifashengguo/STATEToxiCN)
